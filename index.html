使用说明

关于服务

Github Proxy是一个GitHub反向代理服务，通过域名映射技术，将GitHub相关资源请求重定向到镜像服务器，提升访问速度和稳定性。

如何使用

  • 在输入框中粘贴GitHub链接
  • 支持完整URL和简写形式
  • 点击"转换链接"按钮
  • 复制转换后的链接使用
  • 或直接访问转换后的链接

注意事项

  • 支持不带协议的简写链接
  • 部分路径会重定向到本页面
  • 部分高级功能受限
  • 不支持访问私有仓库
  • 请遵守GitHub服务条款
